# team-profile-generator

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

# Description

### This project will allow you to make team profile cards for your business. Using the command line, you'll be asked a series of questions about your team manager, employee's, engineer's, and intern's. Once you complete the series of questions that is being asked, a new HTML page will be generated with the information you provided in the command line.
